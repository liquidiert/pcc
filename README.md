# PythonClientCare

## Installation

### Prerequisites
- Python3.12; can be found [here](https://www.python.org/downloads/)

### Installation steps
1. create new python evironment:
```shell
python3 -m venv env
```
2. activate environment:
```shell
# for linux
. env/bin/activate
# for windows
.\env\bin\Activate.ps1
```
3. install requirements:
```shell
pip3 install -r requirements.txt
```

### Start PCC
Execute:  
`python3 main.py`
